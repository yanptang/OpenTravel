# OpenTravel 行程

**北京** | 2026-04-20 至 2026-04-21
人数: 2 | 交通方式: 公共交通 | 预算: 均衡
必须安排: 故宫, 北京烤鸭

## 第 1 天 (2026-04-20)
**住宿**: 北京

- [1] 08:00-08:45 [T] 交通: 抵达北京站
  - 北京站
  - 从天津乘坐火车到达北京站，办理入住手续。 费用约0元

- [2] 09:00-09:30 [T] 交通: 前往故宫
  - 北京站 -> 故宫博物院
  - 乘坐地铁4号线或公交前往故宫，避开高峰时段。 费用约15元

- [3] 09:30-12:00 [A] 活动: 游览故宫
  - 故宫博物院
  - 参观太和殿、中和殿、保和殿及乾清宫等主要建筑，感受皇家气派。 费用约60元

- [4] 12:00-13:30 [M] 餐饮: 午餐：尝试北京烤鸭
  - 四季民福烤鸭店（故宫店）
  - 品尝正宗北京烤鸭，搭配葱丝、黄瓜和甜面酱。 费用约180元

- [5] 13:30-14:15 [T] 交通: 前往酒店
  - 故宫 -> 北京站附近酒店
  - 乘坐地铁2号线或3号线前往酒店，结束第一天的行程。 费用约10元

- [6] 14:15-15:00 [H] 住宿: 入住酒店
  - 北京站附近酒店
  - 办理入住手续，寄存行李，稍作休息。 费用约0元
